About Terraria

Terraria is kind of similar to Minecraft in that it has blocks that you can mine and other similar mechanics. However, Terraria is a 2D game and so it has different gameplay to minecraft. Terraria has way more things to do that minecraft does. In Terraria there are tons more items and several different bosses. Minecraft, however, only has 2 bosses.
